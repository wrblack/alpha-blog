== README

This is the source to alpha-blog-wrblack.herokuapps.com  
It is a complete blogging sites with the ability to create users, user authentication and authorization.  

* Ruby Version: 2.3.1p11  

* System dependencies: At least Rails 5.0+  

* Database creation and initialization
> rails db:migrate or rake db:migrate

* How to run the test suite  
> rails test

* Deployment instructions  
> Include the PostGres gem for deployment on Heroku.
